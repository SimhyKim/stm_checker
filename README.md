# stm_checker
Check Steam downloading speed in the background 

1) Скачайте и сохраните checker.py в удобную папку (например, C:/Users/admin/Downloads).
2) Откройте командную строку (Win+R - cmd - Enter).
3) Перейдите в папку: `cd C:/Users/admin/Downloads` (или ваш путь).
4) Выполните: `python checker.py`

А еще можно просто скачать и запустить в любой IDE
